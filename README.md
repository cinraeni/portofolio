## 🚀 Tech Stack

This website is built using a modern, fast, and scalable tech stack:

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 3](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics:** [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) & Drei
- **Icons:** Lucide React & React Icons

---

## 🌟 Key Features

- **Interactive 3D ID Card:** A draggable, fully 3D rendered ID card in the Hero section using `@react-three/fiber` and `@react-three/drei`.
- **Dynamic Project Showcase:** A clean and modern project gallery with detailed views for each project, including image sliders and feature lists.
- **Smooth Animations:** Integrated micro-interactions and scroll animations using Framer Motion for a premium user experience.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop views.
- **Static Fallbacks:** Works flawlessly with predefined static data (No database required).

## 🛠️ Getting Started (Run Locally)

If you want to run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/cinraeni/portofolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd portofolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open in Browser:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
